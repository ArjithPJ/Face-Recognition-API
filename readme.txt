Face Recognition and Face Registering API:
-Recognizes faces from videos and webcam from trained images.
-Face Registering API also trains the unknown person's images real-time.

Input:
-Video or Webcam feed.
-Trained images are stored in the faces folder.

Output:
-Person's name.
-If it is not recognized the name of the person must be entered so it can train and recognzize the next time.
